## 亮点功能

- 自动刷理智，掉落物识别
- 自动领取日常奖励
- 自动访问好友基建
- 自动公招，选择使用加急许可 (未来版本)
- 自动基建换班 (未来版本)

## 使用说明

### 基本

1. 请根据 [模拟器支持情况]()，进行对应模拟器选择。
2. 修改模拟器分辨率为 比例 **16：9**,分辨率 **1280x720**。
3. 开始使用！
